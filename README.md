# yus3554.github.io

## 概要

yus3554のポートフォリオサイトです。

## 使用技術

- Ruby
- bundler
- Jekyll
- Jekyllテーマ
  - https://github.com/jeromelachaud/freelancer-theme

## 更新方法

### 基本情報

- _config.htmlを編集する

### ポートフォリオ

- _postsフォルダ内に、日付-プロジェクト名.markdownでファイルを作る
- img/portfolioフォルダ内に追加するプロジェクトの画像をいれる
- markdownファイルを編集する

### About

- _includes/about.htmlを編集する
